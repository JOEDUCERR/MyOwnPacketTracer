# PacketSniffer

This is a basic C program which uses kernel network modules to sniff packets and their size on your linux machine.

## Installation

```
git clone https://github.com/JOEDUCERR/MyOwnPacketTracer.git

chmod +x ./packetsniffer.c

gcc packetsniffer.c -o packetsniffer

sudo ./packetsniffer
```

P.S: Works on all linux distributions.
